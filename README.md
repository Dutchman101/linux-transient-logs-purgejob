Extendable and universal (just add your log folders into the array).
Complexity/unpretty code needed to accomodate as many Linux distro's and systems as possible. As well as for robustness, reliability.

The script is self-contained and will deploy as soon you paste and enter it into your ssh shell (e.g PuTTy)

Pruning interval is set to 3 days by defualt, it won't make exceptions - for server operators that will only visit recent logs when there is a reason to.
